Tasks Description

Here is a list of all tasks which can be spawned by koji. Most of them are available via CLI, some are just child tasks of others.

buildMaven

buildNotification

Koji sends build notifications to the package owner and the user who submitted the build. For BuildNotifications to work successfully, the package owner’s username needs to match a valid username for an e-mail address, because kojihub sends to username@domain in kojihub.conf.

For SSL authentication, this means that your CN must be valid as the user portion of an e-mail address.
